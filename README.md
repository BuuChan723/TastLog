# やさしいMarkdownエディタ

HTML/CSS/JavaScriptだけで動く、初心者向けのMarkdownエディタです。サーバー通信や外部APIは使わず、すべてブラウザ内で処理します。

## ファイル構成

- `index.html`
- `style.css`
- `script.js`
- `README.md`

## 使い方

1. `index.html` をブラウザで開きます。
2. 左側の編集欄に文章を書きます。
3. 文章を選択して、上部のプルダウンから使いたい形式を選びます。
4. 未選択のまま形式を選ぶと、カーソル位置にひな形が入ります。
5. 右側のプレビューで見た目を確認します。

## 主な機能

- 見出し1〜6
- 箇条書き、番号リスト、チェックリスト
- 太字、斜体、太字斜体、取り消し線、文中コード
- 引用、コードブロック、HTMLブロック、明示的な改行
- リンク、画像、表、脚注、区切り線
- リアルタイムプレビュー
- Markdownファイル保存、整形
- 補正結果・チェック結果の表示
- レスポンシブ対応

## ツールバー分類

- 見出し: 大見出し、中見出し、小見出し、見出し4〜6
- リスト: 箇条書き、番号リスト、チェックリスト
- 文字装飾: 太字、斜体、太字斜体、取り消し線、文中コード
- ブロック: 引用、コードブロック、HTMLブロック、改行
- 挿入: リンク、画像、表、脚注、区切り線

## 補足

保存ボタンは、現在のMarkdownを `.md` ファイルとしてダウンロードします。入力内容はブラウザのローカルストレージにも一時保存されます。
